# 📋 Инструкция: Выбор клиента при создании заявки

## Для администратора

### 🎯 Создание заявки с выбором клиента

1. **Откройте бота** и нажмите кнопку **"📦 Создать заявку"**

2. **Выберите склад** из списка

3. **Добавьте товары:**
   - Выберите товар из списка
   - Введите количество
   - Нажмите "➕ Добавить еще товар" или "✅ Продолжить"

4. **Выберите клиента** из списка:
   - Появится список клиентов в виде кнопок
   - Каждая кнопка показывает: `Имя | Телефон`
   - Нажмите на нужного клиента

5. **Завершите создание заявки**:
   - Введите номер транспорта
   - Добавьте комментарий (или отправьте "-" чтобы пропустить)
   - Подтвердите заявку

### 🔍 Поиск клиента

Если в списке много клиентов, используйте поиск:

```
/findclient Имя_клиента
```

**Примеры:**
- `/findclient Алишер` - найдет всех клиентов с "Алишер" в имени
- `/findclient 992` - найдет клиентов с номером, содержащим "992"
- `/findclient Иванов` - найдет клиентов с фамилией "Иванов"

Поиск работает по имени и телефону, регистр не важен.

### ❌ Отмена создания заявки

Если передумали создавать заявку:
- Нажмите кнопку **"❌ Отмена"** в списке клиентов
- Вернетесь в главное меню

## 💡 Полезные советы

### Преимущества нового способа:
- ✅ Сначала заполняете заявку (склад, товары)
- ✅ Затем выбираете клиента из списка
- ✅ Не нужно вводить имя и телефон вручную
- ✅ Нет ошибок в написании имен
- ✅ Быстрый выбор из списка
- ✅ Удобный поиск по большой базе клиентов

### Если клиента нет в списке:
1. Отмените создание заявки (кнопка "❌ Отмена")
2. Перейдите в **"👨‍💼 Панель администратора"**
3. Выберите **"👥 Управление клиентами"**
4. Нажмите **"➕ Добавить клиента напрямую"**
5. Используйте команду:
   ```
   /addclient Telegram_ID | Имя | Телефон
   ```
   Пример:
   ```
   /addclient 123456789 | Алишер Иванов | +992901234567
   ```

## 🔧 Команды администратора

| Команда | Описание |
|---------|----------|
| `/findclient Имя` | Поиск клиента по имени или телефону |
| `/addclient ID \| Имя \| Телефон` | Добавить нового клиента |
| `/editclient ID \| Имя \| Телефон` | Изменить данные клиента |
| `/removeclient ID` | Удалить клиента |

## ❓ Часто задаваемые вопросы

**В: Когда именно нужно выбирать клиента?**
О: После того, как вы добавили все товары и нажали "✅ Продолжить". Сначала заполняете заявку, потом выбираете клиента.

**В: Что делать, если список клиентов очень большой?**
О: Используйте команду `/findclient` для поиска нужного клиента.

**В: Можно ли создать заявку без выбора клиента?**
О: Нет, для администратора выбор клиента обязателен. Это исключает ошибки.

**В: Изменился ли процесс для обычных клиентов?**
О: Нет, клиенты по-прежнему создают заявки через кнопку "🏬 Склад" и их данные подставляются автоматически.

**В: Можно ли изменить данные клиента после выбора?**
О: Нет, после выбора клиента его данные фиксируются. Если нужно изменить, отмените заявку и выберите другого клиента или измените данные через панель администратора.

**В: Можно ли использовать /findclient до создания заявки?**
О: Да, но это будет просто поиск. Для выбора клиента в процессе создания заявки используйте /findclient после нажатия "✅ Продолжить".
