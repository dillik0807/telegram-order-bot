# 🎯 ПРОСТОЕ ОБЪЯСНЕНИЕ: Админ создает заявку за клиента

## ЧТО ИЗМЕНИЛОСЬ?

Раньше:
- Админ создавал заявку → вводил свое имя и телефон

Теперь:
- Админ создает заявку → **выбирает клиента из списка** → заявка создается от имени клиента

## КАК ЭТО РАБОТАЕТ?

### Для администратора:

1. Нажимает "📦 Создать заявку"
2. Выбирает склад
3. Выбирает товары и количество
4. Нажимает "✅ Продолжить"
5. **НОВОЕ:** Видит список всех клиентов с телефонами
6. Выбирает нужного клиента
7. Вводит транспорт и комментарий
8. Подтверждает

### Для обычного клиента:

Ничего не изменилось! Работает как раньше.
