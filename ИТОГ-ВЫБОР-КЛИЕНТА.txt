СЦЕНАРИЙ 1: Администратор создает заявку

1. Администратор нажимает: 📦 Создать заявку

2. Бот загружает список клиентов из БД:
   const clients = await database.getAllClients();
   const activeClients = clients.filter(c => c.is_active === 1);

3. Показывает кнопки:
   ┌─────────────────────────────────────┐
   │ Иван Иванов (+992900000001)         │
   │ Петр Петров (+992900000002)         │
   │ Мария Сидорова (нет телефона)       │
   │ ❌ Отмена                            │
   └─────────────────────────────────────┘

4. Администратор выбирает клиента

5. Бот находит клиента по тексту кнопки:
   const selectedClient = activeClients.find(c => 
     text.includes(c.name) && (c.phone ? text.includes(c.phone) : true)
   );

6. Сохраняет данные:
   data.selectedClientId = selectedClient.telegram_id;
   data.name = selectedClient.name;
   data.phone = selectedClient.phone;
   data.step = 'warehouse';

7. Показывает подтверждение и переходит к выбору склада

8. Администратор продолжает создание заявки

9. При сохранении заявка привязывается к выбранному клиенту:
   let targetUserId = data.selectedClientId || userId;
   const user = await database.getOrCreateUser(targetUserId, ...);

СЦЕНАРИЙ 2: Обычный клиент создает заявку

1. Клиент нажимает: 🏬 Склад

2. Сразу переходит к выбору склада (без выбора клиента)

3. Создает заявку для себя как обычно
