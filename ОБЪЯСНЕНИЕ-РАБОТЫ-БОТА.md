# 🤖 КАК РАБОТАЕТ БОТ ДЛЯ ЗАКАЗОВ - ПРОСТОЕ ОБЪЯСНЕНИЕ

## 📌 ЧТО ЭТО ТАКОЕ?

Это Telegram-бот, который помогает клиентам создавать заявки на товары. Заявки автоматически отправляются в WhatsApp или Telegram группу.

---

## 🎯 ОСНОВНАЯ ИДЕЯ

1. **Клиент** пишет боту в Telegram
2. **Бот** собирает информацию о заказе (склад, товары, количество)
3. **Бот** отправляет готовую заявку в WhatsApp или группу
4. **Администратор** видит заявку и обрабатывает её

---

## 👥 КТО МОЖЕТ ИСПОЛЬЗОВАТЬ?

### 1. Администраторы
- Полный доступ ко всем функциям
- Могут одобрять новых клиентов
- Видят статистику и все заявки
- Могут управлять складами и товарами

### 2. Клиенты
- Создают заявки на товары
- Должны быть одобрены администратором
- Видят только свои заявки

---

## 🔄 ПРОЦЕСС РАБОТЫ

### Шаг 1: Регистрация клиента

```
Клиент → /start → Запрос администратору
                ↓
Администратор → Одобряет или отклоняет
                ↓
Клиент → Получает доступ к боту
```

### Шаг 2: Создание заявки (ПЕРВЫЙ РАЗ)

```
1. Клиент нажимает "🏬 Склад"
2. Выбирает склад из списка
3. Нажимает "🛒 Товар"
4. Выбирает товар
5. Вводит количество (например: "10 шт" или "5 т")
6. Может добавить еще товары (повторить шаги 3-5)
7. Нажимает "✅ Завершить заявку"
8. Вводит ИМЯ (только первый раз!)
9. Вводит ТЕЛЕФОН (только первый раз!)
10. Вводит номер транспорта
11. Вводит комментарий (или "-" если нет)
12. Подтверждает заявку
```

### Шаг 3: Создание заявки (ВТОРОЙ И ПОСЛЕДУЮЩИЕ РАЗЫ)

```
1. Клиент нажимает "🏬 Склад"
2. Выбирает склад
3. Нажимает "🛒 Товар"
4. Выбирает товар и количество
5. Нажимает "✅ Завершить заявку"
6. ИМЯ и ТЕЛЕФОН уже подставлены автоматически! ✨
7. Вводит только номер транспорта
8. Вводит комментарий
9. Подтверждает
```

## 💾 ЧТО СОХРАНЯЕТСЯ?

### В базе данных хранится:

1. **Пользователи (users)**
   - Telegram ID
   - Имя клиента
   - Телефон
   - Роль (admin/client)
   - Дата регистрации

2. **Заявки (orders)**
   - Номер заявки
   - ID клиента
   - Склад
   - Транспорт
   - Комментарий
   - Дата и время
   - Статус (активна/удалена)

3. **Товары в заявке (order_items)**
   - ID заявки
   - Название товара
   - Количество

## 🔑 КЛЮЧЕВЫЕ ОСОБЕННОСТИ

### ✅ Умное сохранение данных
- **Имя и телефон** вводятся только ОДИН РАЗ
- При следующих заявках они подставляются автоматически
- Экономия времени клиента

### ✅ Гибкая система складов и товаров
- Администратор может добавлять/удалять склады
- Администратор может добавлять/удалять товары
- Изменения сразу видны всем клиентам

### ✅ Контроль доступа
- Новые клиенты должны быть одобрены
- Администратор видит кто регистрируется
- Можно отклонить нежелательных пользователей

### ✅ Мягкое удаление
- Заявки не удаляются физически
- Помечаются как "удалённые"
- Можно восстановить при необходимости

## 🛠️ ТЕХНИЧЕСКИЕ ДЕТАЛИ

### Используемые технологии:

1. **Node.js** - основа бота
2. **Telegraf** - библиотека для работы с Telegram API
3. **SQLite** - база данных (файл orders.db)
4. **WhatsApp API** - отправка заявок в WhatsApp
5. **ExcelJS** - экспорт заявок в Excel

### Структура файлов:

```
telegram-order-bot/
├── bot.js              ← Основной файл бота
├── database.js         ← Работа с базой данных
├── whatsapp.js         ← Отправка в WhatsApp
├── admin.js            ← Функции администратора
├── excel-export.js     ← Экспорт в Excel
├── orders.db           ← База данных SQLite
├── .env                ← Настройки (токены, пароли)
└── package.json        ← Зависимости проекта
```

## ❓ ЧАСТЫЕ ВОПРОСЫ

**Q: Сколько клиентов может использовать бота?**
A: Неограниченно! База данных SQLite поддерживает миллионы записей.

**Q: Можно ли изменить список товаров?**
A: Да! Администратор может добавлять/удалять товары через панель.

**Q: Что если клиент ошибся в заявке?**
A: Администратор может удалить заявку, клиент создаст новую.

**Q: Безопасно ли хранить телефоны?**
A: Да, данные хранятся локально в зашифрованной базе данных.

**Q: Можно ли использовать без WhatsApp?**
A: Да! Можно отправлять заявки в Telegram группу.

**Q: Работает ли бот 24/7?**
A: Да, если запущен на сервере или VPS.
