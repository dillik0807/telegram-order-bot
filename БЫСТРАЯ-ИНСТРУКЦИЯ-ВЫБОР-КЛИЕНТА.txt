🎯 ЧТО ДОБАВЛЕНО:
   Теперь администратор может выбирать клиента из списка
   вместо ручного ввода имени и телефона при создании заявки.

───────────────────────────────────────────────────────────────

📱 КАК ИСПОЛЬЗОВАТЬ (для администратора):

1. Создайте заявку как обычно:
   • Выберите склад
   • Добавьте товары
   • Нажмите "✅ Продолжить"

2. НОВОЕ! Выберите способ заполнения:
   
   [👥 Выбрать из списка клиентов]  ← Быстрый выбор
   [✍️ Ввести данные вручную]       ← Для новых клиентов
   [🔙 Отмена]

3. При выборе "👥 Выбрать из списка":
   • Увидите всех клиентов с данными
   • Формат: "Имя (Телефон)"
   • Нажмите на нужного клиента

4. Продолжите создание заявки:
   • Введите номер транспорта
   • Добавьте комментарий
   • Подтвердите заявку

───────────────────────────────────────────────────────────────

✅ ПРЕИМУЩЕСТВА:

   ⚡ Быстро - не нужно вводить данные вручную
   ✓ Точно - нет опечаток в именах и телефонах
   👥 Удобно - все клиенты в одном списке
   🔄 Гибко - можно выбрать ручной ввод

───────────────────────────────────────────────────────────────

👤 ДЛЯ ОБЫЧНЫХ ПОЛЬЗОВАТЕЛЕЙ:

   Ничего не изменилось! Стандартный процесс создания заявки.

───────────────────────────────────────────────────────────────

🚀 ЗАПУСК:

   cd telegram-order-bot
   node bot.js

───────────────────────────────────────────────────────────────

📝 ПРИМЕЧАНИЯ:

   • Работает только для администраторов
   • Показываются только клиенты с заполненными данными
   • Можно в любой момент выбрать ручной ввод
   • Не требует изменений в базе данных

───────────────────────────────────────────────────────────────
