ПРОБЛЕМА: Не могу найти "Volumes" в настройках Railway

═══════════════════════════════════════════════════════════════
  📋 ВАРИАНТ 1: НОВЫЙ ИНТЕРФЕЙС RAILWAY (2024+)
═══════════════════════════════════════════════════════════════

1. Откройте ваш проект на Railway
   https://railway.app

2. Нажмите на ваш СЕРВИС (не проект!)
   Это карточка с названием вашего бота

3. Справа откроется панель настроек

4. Прокрутите вниз до раздела "VOLUMES"
   Или ищите "Storage" / "Persistent Storage"

5. Нажмите "+ Add Volume" или "New Volume"

6. Введите Mount Path: /app/data

7. Нажмите "Add" или "Create"

═══════════════════════════════════════════════════════════════
  📋 ВАРИАНТ 2: СТАРЫЙ ИНТЕРФЕЙС RAILWAY
═══════════════════════════════════════════════════════════════

1. Откройте ваш проект на Railway

2. Нажмите на сервис бота

3. Перейдите в "Settings" (⚙️ иконка)

4. Прокрутите вниз до раздела "Volumes"

5. Нажмите "+ New Volume"

6. Mount Path: /app/data

7. Нажмите "Add"

═══════════════════════════════════════════════════════════════
  📋 ВАРИАНТ 3: VOLUMES НЕТ В ИНТЕРФЕЙСЕ
═══════════════════════════════════════════════════════════════

Если Volumes вообще нет в интерфейсе, возможные причины:

1. ПЛАН TRIAL
   - Volumes доступны только на Hobby плане и выше
   - Решение: Обновите план или используйте PostgreSQL

2. СТАРАЯ ВЕРСИЯ ПРОЕКТА
   - Volumes появились недавно
   - Решение: Создайте новый проект или используйте PostgreSQL

3. РЕГИОН НЕ ПОДДЕРЖИВАЕТ
   - Некоторые регионы не поддерживают Volumes
   - Решение: Используйте PostgreSQL

═══════════════════════════════════════════════════════════════
  ✅ РЕКОМЕНДАЦИЯ: ИСПОЛЬЗУЙТЕ POSTGRESQL
═══════════════════════════════════════════════════════════════

PostgreSQL - это ЛУЧШЕЕ решение для Railway:

ПРЕИМУЩЕСТВА:
  ✅ Легко настроить (1 клик)
  ✅ Автоматические бэкапы
  ✅ Не нужны Volumes
  ✅ Работает на всех планах
  ✅ Production-ready

КАК НАСТРОИТЬ:

1. Railway → Ваш проект → "+ New"

2. Выберите "Database" → "Add PostgreSQL"

3. Railway создаст базу данных автоматически

4. Скопируйте DATABASE_URL:
   - Откройте PostgreSQL сервис
   - Variables → DATABASE_URL
   - Скопируйте значение

5. Добавьте в сервис бота:
   - Откройте сервис бота
   - Variables → Add Variable
   - Name: DATABASE_URL
   - Value: (вставьте скопированное)

6. Установите зависимости:
   npm install pg

7. Я создам адаптер для PostgreSQL

8. Деплой:
   git push

═══════════════════════════════════════════════════════════════
  🔧 ВРЕМЕННОЕ РЕШЕНИЕ: РУЧНОЕ УПРАВЛЕНИЕ
═══════════════════════════════════════════════════════════════

Пока настраиваете постоянное хранилище:

ПЕРЕД ДЕПЛОЕМ:
  1. Отправьте боту: /downloaddb
  2. Сохраните файл .db

ПОСЛЕ ДЕПЛОЯ:
  1. Отправьте боту: /uploaddb
  2. Отправьте сохраненный файл .db
  3. Дождитесь подтверждения

Это сохранит ваши данные между деплоями!
