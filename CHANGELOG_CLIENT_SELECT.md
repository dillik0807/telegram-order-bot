# Изменения: Выбор клиента из списка при создании заявки

## Описание
Добавлена функциональность выбора клиента из списка с помощью inline-клавиатуры при создании заявки администратором.

## Что изменилось

### 1. Новый процесс создания заявки для администратора
- При нажатии кнопки "📦 Создать заявку" администратор сначала видит список клиентов
- Клиенты отображаются в виде inline-кнопок (первые 10)
- Формат кнопки: `Имя клиента | Телефон`

### 2. Поиск клиента
- Добавлена команда `/findclient Имя` для поиска клиента по имени или телефону
- Поиск работает по частичному совпадению (регистронезависимый)
- Результаты также отображаются в виде inline-кнопок

### 3. Обработчики callback
- `client_{telegram_id}` - выбор клиента из списка
- `cancel_order` - отмена создания заявки
- `show_more_clients` - информация о количестве клиентов (пока только показывает)

### 4. Изменения в файлах

#### bot.js
- Изменен обработчик "📦 Создать заявку" - теперь показывает список клиентов
- Добавлена команда `/findclient` для поиска клиента
- Экспортированы дополнительные функции и переменные для использования в admin.js

#### admin.js
- Добавлен обработчик `bot.action(/client_(\d+)/)` для выбора клиента
- Добавлен обработчик `bot.action('cancel_order')` для отмены создания заявки
- Обработчики получают доступ к `orderData` через require('./bot')

## Как использовать

### Для администратора:

1. **Создание заявки с выбором клиента:**
   ```
   Нажать "📦 Создать заявку"
   → Выбрать клиента из списка (inline-кнопки)
   → Выбрать склад
   → Добавить товары
   → Подтвердить заявку
   ```

2. **Поиск клиента:**
   ```
   /findclient Алишер
   → Выбрать нужного клиента из результатов
   → Продолжить создание заявки
   ```

3. **Отмена создания заявки:**
   ```
   Нажать кнопку "❌ Отмена" в списке клиентов
   ```

## Преимущества

✅ Быстрый выбор клиента без ручного ввода
✅ Исключены ошибки при вводе имени и телефона
✅ Удобный поиск по большому списку клиентов
✅ Inline-клавиатура не занимает место в чате
✅ Данные клиента автоматически подставляются в заявку

## Совместимость

- Для обычных клиентов процесс создания заявки не изменился
- Клиенты по-прежнему используют кнопку "🏬 Склад" и вводят свои данные автоматически
- Изменения касаются только администраторов

## Технические детали

- Используется Map `orderData` для хранения состояния создания заявки
- Inline-клавиатура ограничена 10 клиентами для удобства
- Callback data формата: `client_{telegram_id}`
- Поиск работает по полям `name` и `phone` (case-insensitive)
